POP3 :: mixed plist ( resource connection [, integer message ] )


Returns ARRAY if has been successfully recived, FALSE if not.

integer message default value are: 0 (all messages)

Example:






$open = POP3::Connect('mail.host.net', 'username', 'password');

// get only the size of 2'nd message
$size = POP3::pList($open, 2);

echo $size ? $size[2].' Bytes' : 'Error 1';

// get size of each message
$sall = POP3::pList($open);

if($sall){
	echo '<br>All messages:';
	foreach($sall as $num => $val) echo '<br>'.$num.' => '.$val.' Bytes';
}else echo 'Error 2';
