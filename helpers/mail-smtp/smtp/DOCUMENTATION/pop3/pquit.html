POP3 :: boolean pquit ( resource connection )


Returns TRUE if has been successfully recived, FALSE if not.

Send QUIT command and close connection.

Example:






$open = POP3::Connect('mail.host.net', 'username', 'password');

$quit = POP3::pQuit($open);

echo $quit ? 'Success' : 'Error';
