POP3 :: mixed pstat ( resource connection )


Returns ARRAY if has been successfully recived, FALSE if not.

Example:






$open = POP3::Connect('mail.host.net', 'username', 'password');

$stat = POP3::pStat($open) OR die('Error');

echo 'Messages: '.$stat[0].', in total '.$stat[1].' Bytes';
