FUNC :: mixed split_content ( string message )


Return ARRAY if is a mail source message, FALSE if not.

Split the source mail content.

Example:






$msg  = 'From: me@test.net'."\r\n".
	'To: client@domain.net'."\r\n".
	'Subject: Hello World!'."\r\n".
	'Date: Sat, 03 Jun 2006 18:05:35 +0300'."\r\n".
	'Content-Type: text/plain'."\r\n".
	"\r\n".
	'Text message';

//die($msg);

$arr = FUNC::split_content($msg);

if($arr) print_r($arr);
else echo 'Invalid mail message!';
