SMTP :: boolean priority ( [ mixed level ] )


Returns TRUE if has been successfully set, FALSE if not.

mixed level default value are: integer 3

integer level possible values are: 1, 3, 5
string level possible values are: High, Normal, Low

Example:






$mail = new SMTP;

$mail->Priority('High') OR die('error');
