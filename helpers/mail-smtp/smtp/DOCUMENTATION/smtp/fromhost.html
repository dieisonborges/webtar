SMTP :: boolean fromhost ( string hostname, boolean &havemx )


Returns TRUE if has been successfully set, FALSE if not.

This function 'FromHost' is designed for anti-spam/bulk mail prevention on client and/or relay mail delivery.

Notice: one of MX zone for string hostname must have same IP address (external/unique) with this mail sender.

Example:






$mail = new SMTP;

$mail->Delivery('client');

$mail->FromHost('maildomain.net', $havemx) OR die('error');

echo $havemx ? 'Ok' : 'Ok, but doesn't have MX zone!';
