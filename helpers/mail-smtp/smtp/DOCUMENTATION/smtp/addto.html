SMTP :: boolean addto ( string address [, string name ] )


Returns TRUE if has been successfully added, FALSE if not.

string address format are user@hostname

Example:






$mail = new SMTP;

$mail->AddTo('user@example.com', 'Name') OR die('error 1');
$mail->AddTo('client@hostname.net') OR die('error 2');
