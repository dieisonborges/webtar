SMTP :: boolean delcc ( [ string address ] )


Returns TRUE if has been successfully deleted, FALSE if not.

string address default value are: all
string address possible values are: all, user@hostname address format

Example:






$mail = new SMTP;

$mail->AddCc('user@domain.net');

// delete only user@domain.net from Cc
$mail->DelCc('user@domain.net');

// also, you can delete all Cc addresses
$mail->DelCc();
