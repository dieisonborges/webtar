SMTP :: boolean delbcc ( [ string address ] )


Returns TRUE if has been successfully deleted, FALSE if not.

string address default value are: all
string address possible values are: all, user@hostname address format

Example:






$mail = new SMTP;

$mail->AddBcc('user@domain.net');

// delete only user@domain.net from Bcc
$mail->DelBcc('user@domain.net');

// also, you can delete all Bcc addresses
$mail->DelBcc();
