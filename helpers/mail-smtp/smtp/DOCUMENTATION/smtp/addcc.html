SMTP :: boolean addcc ( string address [, string name ] )


Returns TRUE if has been successfully added, FALSE if not.

string address format are user@hostname

Example:






$mail = new SMTP;

$mail->AddCc('user@example.com', 'Name') or die('error 1');
$mail->AddCc('client@hostname.net') or die('error 2');
