SMTP :: boolean port ( integer number )


Returns TRUE if has been successfully set, FALSE if not.

Set the socket port connection to client mail server.

Notice: use this function 'Port' to set port value only if differ from standard 25.

Example:






$mail = new SMTP;

$mail->Delivery('client');

$pset = $mail->Port(100);

echo $pset ? 'ok' : 'error';
